# Octane Discord Bot
Deadly-simple Discord bot to obtain Apex Legends in-game stats, made fully in Rust 🦀 with Serenity 🐈 and Poise 🐢

## Installation
1. Clone the repo
```bash
git clone https://github.com/simxnet/octane
```
2. Fill .env (create one)
3. Run the bot
```bash
cargo run
```
